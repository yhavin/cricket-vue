# Cricket scoring

Keep the score and lose the arguments.

## Features
- Add players and teams
- Score each ball using runs and wickets
- View current score summary
- View full batting and bowling scorecard
- View fall of wicket
- Browse ball-by-ball history
- Save matches
- Export matches
